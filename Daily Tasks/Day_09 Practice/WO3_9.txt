                                         VERBAL ABILITY

Reading Passage

Algorithms are step-by-step procedures used to solve problems efficiently.
Good algorithms reduce time complexity and memory usage.
Efficient algorithms are very important in software development.

Questions

1. What is an algorithm?
Ans:
An algorithm is a step-by-step procedure used to solve problems efficiently.

2. Why are efficient algorithms important?
Ans:
Efficient algorithms are important because they reduce time complexity and memory usage, making software faster and more resource-friendly.

3. What do algorithms help reduce?
Ans:
Algorithms help reduce time complexity and memory usage.
